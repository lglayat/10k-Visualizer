Noteslocker

Introduction

A backend API for an online marketplace for users to buy and sell notes for educational purposes.

* Ruby version 2.3.1

* Rails version 5.1.3

* System dependencies
  - bcrypt
  - jwt
  - rack


* How to run the test suite
  - run 'rails s' in your terminal
  - once up and running, go to http://localhost:3000
